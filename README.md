\# Movie Recommendation System



A simple Python project that recommends movies based on genre.



\## Project Overview



This project uses a small dataset of movies and recommends movies to the user based on their preferred genre.



\## Technologies Used



\- Python

\- Pandas



\## Project Structure



movie-recommendation-system



movies.csv  

movie\_recommender.py  

requirements.txt  

README.md  



\## Dataset



The dataset contains two columns:



title – movie name  

genre – movie category  



Example:



title,genre  

Avengers,Action  

Titanic,Romance  



\## Installation



Install required library:



pip install -r requirements.txt



\## Run the Project



python movie\_recommender.py



\## Example



Enter genre: Action



Recommended Movies:

Avengers

John Wick

Mad Max
